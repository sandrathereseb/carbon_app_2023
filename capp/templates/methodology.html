Methodology

Our carbon calculator uses mathematical formulas from different sources. To calculate your carbon emission, you have to know the distance you travel per day, week, month or year. This calculator is using the target units kilometers and liters, and european standards of carbon emission.

TRAM

An average of 339 grams per person per day.

This is based upon the yearly direct and indirect emissions from all the combined trams divided by the daily amount of users and the days per year. This gives us the following formula:
\[ { { {CO_2 emissions} \over person }\over day }= { { {emission_Y} \over 365 days }\over DailyPassengers }= { { {7417.5tons CO_2} \over 365 days }\over 60 000 } \]

   PLANE

The formula for calculating carbon emissions per person per kilometer for a flight is:
\[ { {CO_2 emissions Per Flight} \over passenger Capacity} * { {1} \over distance }\]
where CO2 emissions per flight is the total amount of CO2 emitted by the aircraft during the flight, passenger capacity is the number of passengers on the flight, and distance is the distance of the flight in kilometers.
To calculate the carbon emissions per person per kilometer, you would then divide this result by the number of passengers on the flight. So the formula is: \[ { { {CO_2EmissionsPerFlight} \over PassengerCapacity} * { {1}\over distance} } \over N.O.Passengers \] It's worth noting that this formula provides only an estimate, and actual emissions can vary depending on factors such as the type of aircraft, load factor, and routing.
